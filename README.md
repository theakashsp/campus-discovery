# Campus Compare

A full-stack college discovery platform built with Next.js and Express.

## Structure

```
frontend/   → Next.js App (UI + internal API routes)
backend/    → Express.js API Server (data + auth)
```

## Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Opens at http://localhost:3000

### Backend
```bash
cd backend
npm install
npm run dev
```
Runs at http://localhost:5000
